Parágrafos, Quebras de linha, Símbolos/Emojis

é possível escrever parágrafos de qualquer jeito desde que estejam dentro de suas tags (<p> e </p>)

entre um parágrafo e outro, a linha é naturalmente quebrada juntamente com um espaçamento
mas pode quebra-la dentro do mesmo usando a tag <br>, como agora.

símbolos especiais:
®, ©, ™, €, £, ¥, ¢, Δ, ↑...

emojis:
🔒, 🔑, 🔓...
